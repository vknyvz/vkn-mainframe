# 🔧 Docker Setup

Centralized **Docker setup** for managing and deploying all my applications.  
This repository provides configurations, automation, and deployment workflows.

---

## 🏗️ Architecture

- **GitHub Actions**: CI/CD pipeline to build and push Docker images.
- **Docker Compose / Services**: Orchestrates multiple applications and APIs.
- **Nginx + SSL**: Handles reverse proxying, subdomains, and HTTPS.
